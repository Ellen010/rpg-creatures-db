<h3>RGB Creatures App</h3>
<p>This app searches for creatures from an RPG game by the name or ID. It displays the results to the user in the table.</p>
<p>Code includes:</p>
<p>-Fetch API,</p>
<p>-Promises,</p>
<p>-Async/Await,</p>
<p>-and the try..catch statement.
